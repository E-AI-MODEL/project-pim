# Project PiM agent instructions

## Product identity

Project PiM is a browser-first, local-first privacy product for educational text. It is a focused product, not a marketing website, technical showcase, dashboard collection, or AI demonstration.

The primary product flow is:

1. enter or open text;
2. detect privacy-sensitive information;
3. create and review a safe version;
4. allow only authorized output actions.
